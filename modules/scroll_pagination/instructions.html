Documentation/Instructions for using scroll_pagination module/extenstion.

##copy scroll_pagination folder in your modules folder.

## put following line into your controller's method and change the required variables .

$this->load->helper('scroll_pagination/scroll_pagination');

$data['scroll_pagination']    = scroll_pagination($base_url,$total_rows,$per_page,$page_segment);

## make sure the list/paging container div id is my_data or change it from library scroll_pagination/scroll_pagination.

##put following line in your view (on the bottom of the page) . Make sure to put this line after jquery call. 

<?php $this->load->view("scroll_pagination/scroll_pagination");?>

Requirements.

  • Jquery must be included. like <script src="http://code.jquery.com/jquery-1.6.1.min.js"> </script>
  • Codeigniter (A PHP based MVC framework) with Moduler Extension (HIMVC)

[Note:-for easy install/avoid permission issues, please put modules folder in parallel of system folder]

Thats it.
